# sistema-pedidos-java
# 🧾 Sistema de Control de Pedidos en Java

## 📌 Descripción
Este proyecto simula el sistema de pedidos de una empresa desde consola.

## 🚀 Funcionalidades
- Registro de cliente
- Selección de productos
- Cálculo de totales
- Aplicación de descuentos

## 🧠 Tecnologías usadas
- Java
- Scanner (entrada de datos)
- Estructuras condicionales (if, switch)


## 📈 Aprendizajes
Este proyecto me ayudó a mejorar mi lógica de programación y entender cómo funcionan sistemas reales.
